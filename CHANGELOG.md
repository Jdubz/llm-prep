# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-02-13

### Added

- Foundations module: transformer architecture, tokenization, attention, context windows, temperature/sampling, embeddings
- Key terminology glossary with ~60 terms across all LLM topics
- Prompt engineering module: techniques guide, common patterns, annotated Python examples
- RAG & embeddings module: core concepts, end-to-end architecture, Python pipeline implementation
- Agents & tool use module: function calling concepts, ReAct/orchestration patterns, Python agent implementation
- Production concerns module: streaming, caching, cost optimization, evals, safety, observability guide with Python examples
- Interview prep module: curated Q&A organized by topic, LLM system design framework
- All code examples in Python using modern patterns (dataclasses, Protocol, async/await, type hints)
