# Hotkey Drills — Build Muscle Memory

Timed editing exercises using real interview code patterns.
Each drill has a starting file and a target. Get from start → target
using ONLY hotkeys — no mouse, no manual character-by-character editing.

## How to practice

1. Open the starter file
2. Start a timer
3. Transform it into the target using hotkeys
4. Check your time against the par

Do each drill 3x. Your third attempt should be under par.

## The hotkeys that matter

| Action | Mac | Linux |
|---|---|---|
| **Multi-cursor (next match)** | Cmd+D | Ctrl+D |
| **Multi-cursor (all matches)** | Cmd+Shift+L | Ctrl+Shift+L |
| **Move line up/down** | Opt+↑/↓ | Alt+↑/↓ |
| **Copy line up/down** | Opt+Shift+↑/↓ | Alt+Shift+↑/↓ |
| **Delete line** | Cmd+Shift+K | Ctrl+Shift+K |
| **Select word** | Cmd+D | Ctrl+D |
| **Select line** | Cmd+L | Ctrl+L |
| **Expand selection** | Cmd+Shift+→ | Ctrl+Shift+→ |
| **Go to matching bracket** | Cmd+Shift+\ | Ctrl+Shift+\ |
| **Add cursor above/below** | Cmd+Opt+↑/↓ | Ctrl+Alt+↑/↓ |
| **Indent/Outdent** | Cmd+]/[ | Ctrl+]/[ |
| **Toggle comment** | Cmd+/ | Ctrl+/ |
| **Go to line** | Ctrl+G | Ctrl+G |
| **Find & Replace** | Cmd+H | Ctrl+H |
| **Quick open file** | Cmd+P | Ctrl+P |
| **Go to symbol** | Cmd+Shift+O | Ctrl+Shift+O |
| **Rename symbol** | F2 | F2 |
| **Trigger suggestion** | Ctrl+Space | Ctrl+Space |
| **Quick fix** | Cmd+. | Ctrl+. |
| **Wrap in...** | Select → Cmd+Shift+P → "Emmet: Wrap" | same |

## Drills

### Drill 1 — Rename & Restructure (par: 60s)
`starter_01.ts` → `target_01.ts`
Skills: F2 rename, Ctrl+D multi-select, Alt+↑↓ move lines

### Drill 2 — Extract Helper (par: 90s)
`starter_02.ts` → `target_02.ts`
Skills: Ctrl+Shift+L select all, multi-cursor typing, copy line

### Drill 3 — Bulk Transform (par: 45s)
`starter_03.ts` → `target_03.ts`
Skills: Find & Replace with regex, Ctrl+D progressive select

### Drill 4 — Build from Types (par: 120s)
`starter_04.ts` → `target_04.ts`
Skills: Ctrl+Space autocomplete, copy line down, multi-cursor edits

### Drill 5 — Refactor Inline to Guard Clauses (par: 75s)
`starter_05.ts` → `target_05.ts`
Skills: Select line, move line, delete line, toggle comment
