# Solutions Engineering, Sales Engineering & Forward Deployed Engineering

How to excel in client-facing technical roles and what to know for SE/FDE interviews. Covers discovery, demos, POCs, technical communication, integration, deployment, and the full customer lifecycle — from first call to expansion.

> **Perspective**: You already know how to build software. This course focuses on the skills that make great client-facing engineers — translating technical depth into customer value, running deals, shipping in someone else's codebase, and thriving at the intersection of engineering and business.

## Content Per Module

Each module directory contains numbered files organized by topic:

- **`01-*.md`** — Foundational concepts and core skills for the topic.
- **`02-*.md`** — Applied patterns, frameworks, and practical techniques.
- **`03-*.md`** — Advanced scenarios, edge cases, and deep-dive content (where present).
- **`exercises.py`** — Hands-on coding exercises with self-test functions. Each exercise has `READ FIRST` and `ALSO SEE` headers pointing to the relevant MD content.
- **`examples.py`** — Complete, runnable reference implementations for each exercise topic.
