# TypeScript Drill Track

Solve these drills in TypeScript first to focus on problem-solving without language ramp-up.

## Drills

| # | File | Class to implement |
|---|------|--------------------|
| 01 | `drill_01_file_storage.ts` | `FileStorage` |
| 02 | `drill_02_key_value_store.ts` | `KeyValueStore` |
| 03 | `drill_03_feature_flag_service.ts` | `FeatureFlagService` |

## How to work

1. Open the drill file and read the spec at the top.
2. Implement the class methods — start with Level 1.
3. Run the file. Self-checks call your methods directly.
4. Fix any `FAIL:` messages, then move to the next level.

Each method stub throws `TODO`. Once you implement it, that error goes away and the self-checks for that level run.

## Run

```bash
# From repo root
npx tsx codesignal-drills/typescript/drill_01_file_storage.ts
npx tsx codesignal-drills/typescript/drill_02_key_value_store.ts
npx tsx codesignal-drills/typescript/drill_03_feature_flag_service.ts
```
