# Python Drill Track

Use this after the TypeScript track, or directly if you prefer Python.

## Drills

| # | File | Class to implement |
|---|------|--------------------|
| 01 | `drill_01_file_storage.py` | `FileStorage` |
| 02 | `drill_02_key_value_store.py` | `KeyValueStore` |
| 03 | `drill_03_feature_flag_service.py` | `FeatureFlagService` |

## How to work

1. Open the drill file and read the spec at the top.
2. Implement the class methods — start with Level 1.
3. Run the file. Self-checks call your methods directly.
4. Fix any `FAIL:` messages, then move to the next level.

Each method stub raises `NotImplementedError`. Once you implement it, that error goes away and the self-checks for that level run.

## Run

```bash
# From repo root
python3 codesignal-drills/python/drill_01_file_storage.py
python3 codesignal-drills/python/drill_02_key_value_store.py
python3 codesignal-drills/python/drill_03_feature_flag_service.py
```
