# Code of Conduct

This project follows the **Contributor Covenant v2.1**.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Summary

We are committed to providing a welcoming and inclusive experience for everyone. We expect all participants to be respectful and constructive in all interactions.

## Reporting

If you experience or witness unacceptable behavior, please open an issue or contact the maintainers directly.

## Enforcement

Project maintainers will take appropriate action in response to any behavior that violates this code of conduct.
